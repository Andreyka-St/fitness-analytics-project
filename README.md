# Проект: Анализ эффективности фитнес-клуба

## Описание
Анализ данных о клиентах, посещениях и маркетинговых кампаниях фитнес-клуба.

## Использованные технологии
- Python (pandas, matplotlib, seaborn)
- Jupyter Notebook
- SQL (PostgreSQL)

## Структура репозитория
- `fitness_project.ipynb` — основной анализ.
- `clients`, `visits`, `marketing`  — CSV-файлы с данными.
- `README.md` — описание проекта.

## Выводы
- VIP-абонементы приносят наибольшую выручку.
- Наиболее эффективная кампания — "Новогодняя акция".
- Групповые абонементы являются самыми популярными.
